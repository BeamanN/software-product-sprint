Image Processing Matrix Operations

Project for Programming Methodology Class where we had to complete an image processing library that can process images using matrix operations. All images can be represented by a matrix of single-byte values, where a pixel at matrix location (i, j) is represented by a number between 0 and 255. A color image consists of three channels: a red channel (R), a green channel (G), and a blue channel (B), and the corresponding matrix information for an image includes three matrixes, each with the same number of rows (height) and columns (width), one foreach of the three channels. Together,these are passed to an image-writing library for outputting to a file and displaying.
